Installation
------------

Releases are hosted on PyPI_. To install tomial-clicky-tooth, run
the following in your terminal:

.. code-block:: console

    pip install tomial-clicky-tooth

.. _PyPI: https://pypi.org/project/tomial-clicky-tooth/
